0. 约定/规律

- 涉及密码的用POST，其他的用GET
- status 200为成功，40x为用户错误/输入错误，50x为服务器错误
- 单个对象类数据在data字段中，多个/数组在data_set字段中

例子：

````json
{
	"status": 200,   /* 状态码 */
	"msg": "",   /* 错误信息 */
	"data": {
		"result": true,  /* 返回结果 bool */
		"name": "手机用户123678113",  /* 临时用户名 string */
		"token": "ADASDASZXC"  /* 用户鉴权的token令牌 string */
	}
}
````

````json
{
	"status": 200,   /* 状态码 */
	"msg": "",   /* 错误信息 */
	"data_set": [
		1, 
		2, 
		3, 
		4,
		5
	]
}
````
